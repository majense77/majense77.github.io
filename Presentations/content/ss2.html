Sketch Sessions


It's time to get sketchy.

But don't worry - you don't need to be a pro artist to be able to participate. The point here is to provide an outlet for translating "How Might We" (HMW) oppontunities into potential solutions.

3 Box, 3 Line

  • Rules

    Pretty simple:

    1) Each box has a sketch.

    2) Each line has a word.

  • Guidelines

    1) Sketches don't need to be "good", just able to communicate an idea.

    2) Separate box/line combos can be separate entirely, or they can relate to one another.

    3) You can draw more box/line combos if you need to - 3 is just the lower limit!
